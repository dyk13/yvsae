슈퍼스타 Tags

Document Type Declaration: 어떤 version의 html을 썼는지 알려준다. (!doctype html)
html: HTML 문서의 시작과 끝을 알려준다.
head: 본문의 제목이라든지 어떤 문자 형식으로 작성되었는지 등 설명을 해준다.
title: 웹페이지의 제목. 웹브라우저의 맨 위 tab에 뜬다.
meta: metadata. 자료에 대한 정보를 제공한다. 예를 들어 (meta charset="utf-8") 이라고 되어 있다면 "이 문서는 utf-8이라는 방식으로 저장되어 있다"라는 의미이다.
body: 본문으로 말하고자 하는 내용이 들어간다.
